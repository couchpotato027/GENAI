About This Project

Intelligent Exam Question Analysis — Milestone 1

This project is a machine learning-based analytics system that predicts exam question difficulty using classical ML techniques. It accepts exam question text along with student response data, extracts text and numeric features using TF-IDF vectorization and statistical measures, and classifies question difficulty as Easy, Medium, or Hard.
